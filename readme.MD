## Installation

To install and run the game, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/yourusername/connect-4.git
```

2. Navigate to the project directory:

```bash
cd connect-4
```

3. Install dependencies:

```bash
pip install -e .
```

4. Run the game:

```bash
python connect4.py
```
